# SchoolSystem

## Features

1. Webserver und APIs mit Expressjs 
1. Anmeldung und Registrierung mit Passportjs
1. Nutzer, Kurs, und session Daten in Postgres Sql

## Einrichtung

NodeJS sollte installiert sein und eine PSQL Datenbank sollte eingerichtet sein

1. Dieses repo klonen 
```git
git clone https://github.com/Eyup3/SchoolSystem.git
```

2. NodeJS Pakete installieren
```
npm install
```
3. .env Datei einrichten \
Die .env Datei je nach Situation einrichten

4. Das Programm starten

```
npm run dev
```
